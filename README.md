# Shifankhan Pathan — Portfolio

A clean, modern, fully responsive **MERN Developer Portfolio** built using **React, Tailwind CSS, and Framer Motion**.  
Showcases my projects, skills, resume, and contact information in a minimal and professional design.

Live Demo **https://shifan-portfolio.vercel.app**

---

## Features

-  Smooth animations with Framer Motion  
-  Minimal and modern UI using Tailwind CSS  
-  Fully responsive on all devices  
-  Project showcase with screenshots  
-  Downloadable PDF Resume  
-  GitHub & LinkedIn links  
-  Organized sections (Hero, Projects, Skills, About, Contact)

---

## Tech Stack

**Frontend:**  
- React  
- Tailwind CSS  
- Framer Motion  

**Build Tool:**  
- Vite (or CRA if using default setup)

**Deployment:**  
- Vercel  

---

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm start
```

### Build for production
```bash
npm run build
```
